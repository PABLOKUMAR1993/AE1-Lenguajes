Que hace CoolPic. para ti?

Es una página web para buscar insparación, para fotografia, dibujo, make-up, arte en general...
La página de inicio tiene una galería de 20 fotografías aleatorias, de cualquie tematica, cada vez que recargues la web cambiarán.
Si necesitas algo concreto puedes usar el buscador y te devolverá 10 fotografias por el término de busqueda introducido
